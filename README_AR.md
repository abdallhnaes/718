# ديربي كفردريان V8 - مديرين وصلاحيات

## الجديد
- تعديل واجهة دخول المدير:
  - بوابة إدارة منظمة
  - اختيار نوع المدير
  - دخول المدير الرئيسي
  - دخول مدير الفريق الثاني

## رموز الدخول
- المدير الرئيسي: 5135
- مدير الفريق الثاني: 2222

## صلاحيات المدير الرئيسي
- تعديل الفريق الأول
- تعديل الفريق الثاني
- تحديد موعد المباراة القادمة
- تسجيل النتائج
- مسح السجل
- رؤية كل الخطط حتى إذا كانت مخفية عن المستخدمين

## صلاحيات مدير الفريق الثاني
- تعديل كل ما يخص الفريق الثاني فقط
- تعديل اسم الفريق الثاني
- تعديل لاعبي الفريق الثاني
- تعديل البديل
- اختيار خطة جاهزة أو رسم خطة يدوية بالسحب
- منع عرض تشكيلة الفريق الثاني للمستخدمين

## المستخدمون
- يفتح التطبيق مباشرة على الواجهة الداخلية
- يرى موعد المباراة القادمة
- يرى خطة الفريق الأول
- يرى خطة الفريق الثاني فقط إذا لم يمنع مدير الفريق الثاني عرضها
- يرى النتائج والمباريات السابقة
- لا يستطيع التعديل

## ملاحظة
هذه النسخة محلية على نفس الهاتف. لو تريد مدير من جهاز والمستخدمين من أجهزة ثانية، نحتاج نسخة Firebase.
